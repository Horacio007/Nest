# Teslo API
1. Clonar el repo
2. Ejecutar 
```
npm install
```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno 
5. Levantar el modo desarrollo ```npm run start:dev```
